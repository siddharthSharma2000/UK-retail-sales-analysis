# UK Retail Sales Analysis

## Business Context
A UK-based retail company wants to understand its sales performance across different regions and product categories to support better commercial decision-making.

## Objective
The objective of this project is to analyse retail sales data to:
- Identify high and low performing regions
- Understand product category performance
- Compare weekday and weekend sales trends

## Tools Used
- Excel (data cleaning and preparation)
- SQL (data analysis and querying)
- Tableau (data visualisation and dashboards)

## Status
This project is currently in progress.
